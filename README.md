## Description

Pullr is a Ruby library for quickly pulling down or updating any Repository.
Pullr currently supports Git, Mercurial (Hg), SubVersion (SVN) and Rsync.

## Features

* Currently supports pulling from:
  * Mercurial (Hg)
  * Git
  * SubVersion (SVN)
  * Rsync
